# Skills

SKILLS 是可复用的 AI 能力集合，帮助开发者更高效地完成日常任务。

## 代码导航

- `/find-module` - 查找模块定义和引用
- `/explain-class` - 解释类的职责和依赖
- `/trace-call` - 追踪函数调用链

## 开发工作流

- `/create-feature` - 创建特性分支和模板
- `/fix-bug` - 分析和修复 Bug
- `/submit-pr` - 提交 Pull Request

## 文档生成

- `/generate-api-doc` - 生成 API 文档
- `/create-diagram` - 创建架构图
