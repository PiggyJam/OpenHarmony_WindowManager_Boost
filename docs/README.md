这是一个OpenHarmony WindowManager Subsystem AI辅助仓库。

WindowManager是OpenHarmony和HarmonyOS的公共窗口管理服务底座。
代码仓库：https://gitcode.com/openharmony/window_window_manager
DEEPWIKI：

## 目录结构
1. `wiki/`：是对整个窗口管理服务的架构、模块、特性的详细解读，帮助开发者了解OH窗口管理子系统的设计和实现。
2. `agentic/`: 集成了一些OH窗口管理子系统从设计、开发、测试、编译等等过程的一些AI辅助流程，主要是一些SKILLS集合、Agent集合，以后可能支持..？(暂时不知)


## 静态网站
1. 为了方便阅读，本仓库提供Github Pages托管的静态网站。
2. 当然也提供参与贡献的方式，自动集成并通过Github Pages展示。
